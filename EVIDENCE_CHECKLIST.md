# Чек-лист доказательств для сдачи

## GitHub

- [ ] Репозиторий публичный или доступен преподавателю.
- [ ] В репозитории есть README.md.
- [ ] В репозитории нет `.env` с реальными секретами.
- [ ] Есть `.env.example`.

## Демо-видео 2–3 минуты

- [ ] Показан запуск сервиса.
- [ ] Показан запрос к `POST /triage`.
- [ ] Показан JSON-ответ.
- [ ] Показана запись в SQLite.
- [ ] Показаны логи.
- [ ] Кратко объяснен fallback.

## Видео защиты 5–7 минут

- [ ] Объяснена проблема бизнеса.
- [ ] Объяснены ограничения MVP.
- [ ] Показана архитектура.
- [ ] Объяснены риски и надежность.
- [ ] Дан план внедрения.

## Скрины

- [ ] Запрос в Postman/curl.
- [ ] Ответ сервиса.
- [ ] SQLite-запись.
- [ ] Логи.
- [ ] GitHub README.
- [ ] Лог откликов.

## Лог откликов

- [ ] 60 строк заполнены.
- [ ] Есть ссылки/скрины.
- [ ] Есть статусы.
